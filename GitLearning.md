# Git學習

## git的配置

### 三個等級的配置

* 針對系統上所有用戶的配置

  `git config --system`


* 針對當前用戶的配置

  `git config --global user.name "LymanChen"`

  `git config --global user.email lyman.chen@astasys.com`

* 針對當前repository的配置

  `git config --local xxxxx`

  *低級別的配置會優先適用*

### 檢查配置信息

* 列出所有配置

  `git config --list`

* 檢查某一項配置

  `git config 配置項`
